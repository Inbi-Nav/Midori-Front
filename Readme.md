#  Midori Frontend

Frontend desarrollado para consumir la API REST del proyecto **Midori**.

Este proyecto forma parte de una práctica enfocada en el desarrollo de frontend mediante el uso de la IA, donde se documenta el proceso completo de diseño, generación de código, análisis y conexión con el backend.
